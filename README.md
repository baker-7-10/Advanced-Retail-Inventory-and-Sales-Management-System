# Advanced Retail Inventory and Sales Management System

Backend built with NestJS, TypeORM, MySQL, and Docker.

## Setup

```bash
npm install
docker compose up -d
npm run build
```

## API Docs
```
http://localhost:3000/api/docs
```

## Tests

```bash
npm test          # unit tests
npm run test:e2e  # e2e tests
npm run test:all  # all tests
```
